# Policy · Defaults

## P1 — Ускорение и кисть (GS) · RESOLVED → modern

**Default:** [[gradual-acceleration]] + [[effortless-power]] — набор скорости к контакту; relax → firm у контакта; без late «boom» / muscling.

**Ок как дополнение:** [[ground-force]] / цепь (Джумок «слияние» энергии замаха + опоры на контакте) = timed peak через ноги→корпус, **не** оправдание рывка и постоянной силовой жёсткости кисти.

**Не default:** Николаев «рывковое заключительное наращивание» + силовая жёсткость кисти на сильных ударах — не цитировать боту как норму.

## P2 — Стойки в обучении · RESOLVED → situational

- Закрытая / нейтральная полезна для освоения [[ground-force]] и линейного transfer.
- Open / semi **не запрещать**, если есть leg drive и [[unit-turn]].
- Бот не говорит «только закрытая сначала» и не толкает early-open как догму.
- Детям / новичкам можно мягко предложить начать с нейтральной/закрытой для чувства опоры.

## P3 — Стойка и мощность (baseline) · RESOLVED → situational

- Стойка = время + геометрия мяча.
- Open / semi — полноценная атака при активном [[ground-force]] / [[hip-rotation]].
- Закрытая — не единственный путь к силе.
- Запрет абсолютов: «open = только защита» и «сила только из закрытой».

## 2HBH grip · RESOLVED

- **Default:** bottom/dominant Continental + top/non-dom Eastern FH.
- Semi-Western top **допустим**, если игрок уже так играет и topspin/высота мяча это оправдывают — не ломать рабочий grip без причины.
- См. [[backhand-2h]] · [[grip]].

## Как писать в ответах бота

1. По ускорению/кисти — всегда P1 (modern).
2. По стойкам — P2/P3 situational (выше); без догматичных абсолютов.
3. При «жёсткая кисть / рывок» — переводить на gradual + firm-at-contact + ноги.
4. 2HBH grip — Eastern default; SW top только если уже работает у игрока.
