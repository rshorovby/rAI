# Two-handed backhand

## Fundamentals (synthesis)

1. **Grip (policy):** dominant (bottom) Continental; non-dom (top) **Eastern** по умолчанию. Semi-Western top допустим, если уже рабочий и оправдан topspin/высотой — не ломать без причины ([[policy-modern-defaults]]).
2. **Роли рук:** non-dom ≈ motor (power / topspin); dominant ≈ face / control.
3. **Prep:** [[unit-turn]] — поворот корпуса; руками short lift (часто чуть вперёд), не большой arm-backswing ([[visual-illusions]]).
4. **Цепь:** ноги → таз/hips → корпус → руки ([[ground-force]] · [[hip-rotation]] · [[weight-transfer]]).
5. **Contact:** впереди, ~пояс; late/too close → потеря transfer + нагрузка на суставы.
6. **Path:** compact; Verdasco-модель — почти одинаковый path назад/вперёд; loop/drop часто «случается», не отдельная команда.
7. **Finish:** продолжение ротации; ракетка часто over dominant shoulder.

## Power killers (Feel Tennis)

- Наклон корпуса/головы (tilt)
- Нет swing в takeback
- Нет [[hip-rotation]]
- Зажатые кисти / горизонтальный mental image

## Академия / academic notes

- Mouratoglou: акцент shoulder turn + back→front weight transfer; варианты topspin / slice / flat.
- Petrovici 2025: 2HBH стабильнее и безопаснее unilateral load, чем 1HBH; open/semi-open для темпа.
- Джумок: 2HBH стабильнее/сильнее; 1HBH вариативнее ([[backhand]]).

## Notes

- Weight transfer vs rotation: оба = ноги+корпус, не arms-only (FT + Mouratoglou).

## Чеклист разбора

1. Grip change рано; non-dom сверху?
2. Unit turn без большого arm pull?
3. Hips включаются (не push руками)?
4. Контакт впереди / не late?
5. Нет сильного tilt / зажатых кистей?
6. Finish не обрывается?
