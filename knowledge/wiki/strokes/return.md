# Return

## Джумок 2020 (unverified)

- Короче замах, чем обычный GS: прямой / блокирующий.
- Часто **открытая** стойка; контакт может быть в прыжке.
- Timing: шаги вперёд на toss → [[split-step]] на выносе подающего → короткий turn → удар → приземление внутрь на разноименную.
- Позиция у BL×sideline; внутрь = атака, наружу = защита.
- Return «как обычный GS» заранее ставит в защиту.

2. Замах укорочен под скорость?
