# Forehand

## Николаев 2012 (дополнение)

- Контакт **впереди-сбоку**; path под высоту мяча; тенденция к спрямлению замаха.
- Удар с нарастающей скоростью; маскировка направления.
- Ускорение/кисть: default = [[gradual-acceleration]] ([[policy-modern-defaults]] P1), не Николаев «рывок».

## Джумок 2020

- Max энергия — закрытая / полузакрытая; open чаще защита.
- Вход через толчковую («вход в мяч»); ошибка — поворот вокруг себя без опоры стопы.
- Замах под высоту мяча; Semi/Western (Eastern) для topspin/flat.

## 8 steps (Feel Tennis, unverified)

1. Semi-western + index support under handle
2. Ready **state** (движение) + [[split-step]]
3. Prep: turn; off-hand готовит ракетку «в сторону»
4. Drop on the **edge** (не pro face-down flip для любителей)
5. Hips first → wrist lag сам
6. Swing path «bowling» / прямой участок через контакт
7. Compress & roll + extension из [[clear-intention]]
8. Catch ракетку non-dominant рукой
