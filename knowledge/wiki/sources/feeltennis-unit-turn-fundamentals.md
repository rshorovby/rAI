# Source

**URL:** https://www.feeltennis.net/unit-turn/

## Key claims

1. Unit = **torso** (~90°), не toe-to-head.
2. Ошибка: step back → вес назад → потеря силы/времени (на easy ball pivot + step forward).
3. Ошибка: голова уезжает с плечами → нет 3D depth, баланс, jerky rotation.
4. Drill: зеркало — turn torso, голова почти на месте, оба глаза на «мяч».
5. Deep ball — step back естественно ок.
