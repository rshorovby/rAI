## Key claims

1. Два типа short: чуть внутри BL vs около service line.
2. Adjusting: TURN → STEP → SHUFFLE → hit (не бежать без unit turn).
3. Really short: RUN → TURN along way → STOP → hit.
4. Drill patterns без мяча → subconscious; иначе rush/awkward → miss winners.
