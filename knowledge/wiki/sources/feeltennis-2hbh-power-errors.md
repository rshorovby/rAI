# Source · 2HBH power errors

**URL:** https://www.feeltennis.net/tennis-two-handed-backhand-errors/

## Key claims

1. Типичный любительский 2HBH — «pushed», безопасный, без мощности; лечится устранением ошибок, не «большим усилием».
2. **Excessive tilt** головы/корпуса → нельзя вращаться вокруг оси → потеря power + mishits.
3. **No swing** в takeback — просто «поставить ракетку назад» → нет разгона вместе с ротацией.
4. **No hip rotation** — самая частая ошибка; без hips нет effortless power (drill в курсе).
5. **Tight wrists** — горизонтальный mental image → muscling, short balls; нужны живые кисти/руки и дуга.
