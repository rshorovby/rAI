# Source · Return

## Key claims

1. Приём критичен: плохой return → почти всегда проигрыш очка; скорости 2-й подачи уже на грани зрительно-моторных реакций.
2. Современная структура **отличается** от обычного GS сильнее, чем считали классики: короткий (прямой/блокирующий) замах; мгновенная передача импульса ног→ракетка; контакт часто **в прыжке**.
3. Позиция: обычно у пересечения BL и singles sideline ±0.5 м по BL; по sideline 0.5–1 м внутрь … 3–4 м наружу — зависит от 1/2-й, спина, покрытия, тактики, стиля.
4. Ближе к центру/внутрь → атакующий return; шире наружу → защитный.
5. Оптимально из **открытой** стойки (дефицит времени).
6. Timing: 2–4 шага вперёд на toss/замахе подающего → разножка на активном выносе → короткий unit turn + block/straight swing → контакт → приземление внутрь корта на разноименную ногу.
7. Return «как обычный GS» заранее ставит в защиту; описанная структура — путь к атаке/контратаке.
