## Key claims

1. Техника опирается на **natural biomechanics**, не на мнения про «стиль» форхенда.
2. **Парадокс:** сильнее зажимать руку / больше «силы» рукой → хуже мощность и контроль; нужно **меньше** работы рукой, больше ног/корпуса, позволить ракетке **swing**.
3. Фундамент (пример FH / 1HBH): **hip rotation + weight transfer + pendulum swing**.
4. Техника = минимальная модификация фундамента, чтобы не «убить» свинг (throw vs push ratio).
5. Чувство тайминга усиления свинга — через тело (как на детских качелях), не через словесную инструкцию.
6. Для обучения нужны **лёгкие мячи / free hitting**; матчи ломают ощущение свинга.
7. Упражнения на swing, gravity, centrifugal force — чтобы «разбудить» тело после сидячего образа жизни.
