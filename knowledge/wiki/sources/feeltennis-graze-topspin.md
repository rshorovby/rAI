## Key claims

1. Advanced: **graze** (touch lightly in passing) ≠ brush that slows racket.
2. Inside court / short ball: more graze, less full; behind baseline: more into ball + some spin.
3. Precision skill: ratio flat vs spin vectors.
4. Mini tennis для precision feedback.
5. Compress&roll остаётся для новичков.
