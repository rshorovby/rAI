# Source · Groundstrokes

## Key claims

1. GS + serve = фундамент; GS ~68% действий HQ (в основном return + BL FH/BH).
2. Max энергия — закрытая / полузакрытая стойка; open — чаще защита.
3. FH grips: Semi/Western (и Eastern) для topspin/flat; Continental для slice.
4. Фаза 1: замах + unit turn ⊥ сетке + вес на заднюю; высота мяча меняет финиш замаха (высокий / плечи / низкий ~25° отведения плеча).
5. Фаза 2: вход через толчковую ногу («вход в мяч»); стопа опоры ~40–60° к линии мяча ограничивает over-rotation.
6. Главная ошибка: **поворот вокруг себя** без активного взаимодействия стопы с опорой → ломается цепь импульса.
7. Slice: таз/плечи остаются || линии удара (меньше ротации, чем flat/topspin).
8. 2HBH стабильнее и сильнее; 1HBH вариативнее и менее читаем.
9. Flat / topspin / slice — траектории, отскок, покрытия (как у Николаева/FT, с акцентом на topspin как доминирующий GS).
10. Slice: скользящий vs рубящий; чаще защита / recovery / короткий замах; сильный стелющийся — давление.
