# Source

**URL:** https://www.feeltennis.net/visual-learning/

## Key claims

1. Глаза следят за ракеткой → oversized swings → late.
2. Illusions: contact не «в сторону»; path не горизонтальный (bowling); prep рука не высоко; нет wrist slap; slice face не открывается на контакте; volley не away — diagonally across при 45° теле.
3. Убирать ракетку из руки → видеть реальное движение руки.
4. ~70% учатся визуально — правильный mental image критичен.
