# Aim calibration

Если удар стабильно уходит в сторону на X — целиться на X в противоположную сторону («калибровка прицела»), а не каждый раз ломать механику.

## Claims

- Accuracy ≠ precision; serve точнее groundstrokes.
- «Согни себя (aim), не двигатель (stroke)» ([[feeltennis-accuracy-calibration]]).
- Работает при систематическом bias; при хаотичных промахах сначала timing/[[form-vs-technique]].

## Для бота

Если пользователь жалуется «всегда вправо / в сетку» при терпимой механике — предложить calibrate aim + проверить [[clear-intention]] / depth.
