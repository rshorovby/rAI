# Visual illusions

Ложные mental images от пути головки ракетки → слишком большие свинги → [[late-contact]].

## Частые illusions (Feel Tennis)

1. Contact «в сторону» → на самом деле рука больше вперёд
2. Path горизонтальный → скорее up/down (bowling)
3. Prep рука высоко → рука ~shoulder height
4. Wrist slap → face доворачивается от arm/body
5. Slice face открывается на контакте → на контакте почти perpendicular
6. Volley away → diagonally across при теле ~45°
7. **BH/2HBH backswing:** ракетка «назад руками» → на деле turn + short lift; у 2HBH руки часто чуть вперёд ([[feeltennis-backhand-illusions]], [[feeltennis-verdasco-2hbh]])
