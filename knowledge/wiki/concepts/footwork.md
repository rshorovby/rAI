# Footwork

## Цикл (Николаев / Джумок)

Старт → к мячу → вход в удар → выход из удара → [[mezhdudarnoe-vremya]] → позиция (биссектриса).  
В быстром темпе часто **3 компонента** без идеального recovery к центру ([[dzhumok-2020-movement-cycle]]).

Ready: пружинящие колени, ракетка перед поясом. Предпочтение — удар **в движении на мяч**.

Классика: на контакте впереди разноименная нога; также open / выпад / прыжок.

## Джумок 2020

- HQ: бег ~23% / шаги ~37% / прыжки ~40%; среди шагов много семенящих.
- Закрытая стойка — сила и проще баланс; открытая — время, мало мощности; полуоткрытая — частый HQ-выбор.
- Детям рано open/semi-open → выпадает толчковая стопа из цепи (tension с modern open-heavy coaching).

## Short balls (Feel Tennis)

- Чуть внутри BL: **turn → step → shuffle → hit**
- Около service line: **run → turn along way → stop → hit**

## Stances

Neutral/open ← push off L или R leg.

**P2/P3 situational** — см. [[policy-modern-defaults]]: без абсолютов «только закрытая» / «open = защита».
