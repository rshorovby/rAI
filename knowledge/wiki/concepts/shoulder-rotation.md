# Shoulder rotation

Fundamental #3: плечи как unit — сила + спокойный угол ракетки.

## Claims

- FH: «мёртвая» non-dominant рука = disconnect.
- 1HBH/slice: ротация есть (~45°), потом stop — не блок sideways.
- Без ротации: только рука → fatigue + spraying face angle.
- Engage оба плеча equally.
